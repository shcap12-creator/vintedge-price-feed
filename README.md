# Vintedge 価格フィード（対応モデル相場表）

iOSアプリ **Vintedge**（現物資産ポートフォリオ）が読み込む、対応モデルの参考相場データです。

- `catalog.json` — 対応モデルと参考価格の一覧（運営が手動更新）
- すべての価格に **出典（sourceName）・補足（sourceNote）・基準日（asOf）** を付けています
- 価格は**参考情報**であり、実際の売買価格・換金性を保証するものではありません

## 更新の仕方
1. `catalog.json` の該当モデルの `priceYen`（円・整数）と `asOf`（基準日）を書き換える
2. 新しいモデルを追加する場合は `models` 配列に同じ形式で1ブロック追加する
3. ファイル全体の `updatedAt` を更新してコミットする
4. アプリは起動時（12時間に1回）に自動で読みに来ます

## 形式
- `category`: `watch` / `tradingCard` / `wine` / `sneaker`
- `priceYen`: 円・整数（小数は使わない）
- `sampleCount`: 参考にした取引件数（わかる場合のみ）

このリポジトリには市場の参考価格のみが含まれ、ユーザーの資産データは一切含まれません。
